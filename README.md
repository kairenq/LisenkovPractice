# Учет расходуемого материала (простая локальная версия)

Это **простое красивое приложение без БД и без Electron-сборки**.
Все данные хранятся локально в `localStorage` браузера.

## Что есть
- Авторизация и роли (админ, кладовщик, бухгалтер, руководитель)
- Материалы (добавление + список)
- Приход/расход с проверкой остатка
- Таблица остатков с подсветкой низкого остатка
- Отчет за период + печать/PDF
- Пользователи (вкладка админа)
- Логи действий

## Запуск после скачивания архива
```bash
npm install
npm run dev -w @app/desktop
```

Открой адрес из консоли (обычно `http://localhost:5173`).

## Сборка (просто статическая)
```bash
npm run build -w @app/desktop
```
